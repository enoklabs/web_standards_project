Project Analysis

Client: Top Stone Inc.

Project: Website Redesign

Designer: Rey Madrid

Site Intent

The objective of this project is to examine and find the appropriate redesign approach for the website: www.topstone1.com owned by Top Stone Inc.


During the analysis process, I find several areas that need improvement. The website is flat and confusing, therefore lacking proper presentation to the user. There is no layout structure and content is disorganized. The results of the W3C’s validation test showed 83 errors and improper html semantics. Visual design is weak and inconsistent across all pages allowing me to make room for a new and refreshing user experience and overall redesign.

For a better website’s user interface/experience and overall higher conversion rate, my redesign process will consist of the following changes:

  1. Organize the entire website’s content into 6 web pages.
  2. Add a logo and a navigational menu at the top of all pages.
  3. Make a homepage CTA with a clear statement of the website offers.
  4. Teach visitors what is Top Stone Inc. with a brief summary.
  5. Promote the physical store’s location and display their catalog.
  6. Turn online visitors into potential store walk-in customers.
